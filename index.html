Documentación de comandos en consola

El comando "ls" list, muestra los archivos y carpetas

el comando "cd" change directory cambia al directorio

el comando "mkdir" make directory crea un nuevo directorio

Con el comando "ls" observamos el nuevo directorio

Se crea con el comando "touch" nuevamente un archivo

El comando "mv" move mueve un archivo a otra carpeta

El comando "rm" remove elimina un archivo

Con el comando "touch" se crea nuevamente el archivo index.html

El comando "nano" permite editar archivos desde la consola

El comando "vim" también permite editar archivos desde la consola

Se crean carpetas "mkdir" y archivos "touch"

El comando "ls -a" muestra los archivos y las carpetas con detalle

El comando "ls-l" muestra los archivos con los permisos

Se cambian los permisos del archivo README.md
